## ローカル環境でのあそびかた

git clone する

``` bash
$ git clone https://github.com/c5meg1012/vue-flag.git
```

npm install する


``` bash
$ npm install
```

ローカルでサーバーを立ち上げる

``` bash
$ npm run dev
```

以下のURLが立ち上がります

http://localhost:8080/
